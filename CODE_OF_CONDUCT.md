# Code of Conduct

AgentDeck 社区希望保持专业、直接、尊重和以问题解决为导向的协作环境。

## 期望行为

- 尊重不同背景和经验水平的贡献者。
- 聚焦技术事实、产品目标和用户价值。
- 对设计和实现提出具体、可验证的反馈。
- 在讨论安全问题时优先保护用户和项目。
- 接受维护者对范围、优先级和实现方式的取舍。

## 不接受的行为

- 人身攻击、骚扰、歧视或威胁。
- 故意发布 secret、漏洞利用细节或破坏性命令。
- 在维护者明确拒绝后持续推动无关议题。
- 垃圾信息、广告或与项目无关的推广。

## 处理方式

维护者可以编辑、隐藏或删除违反准则的内容，也可以限制参与者继续互动。严重安全或骚扰问题会优先处理。

## 适用范围

本准则适用于 GitHub issue、pull request、discussion、release、文档和其他与 AgentDeck 项目相关的公共协作空间。
